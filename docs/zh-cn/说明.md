# docs/zh-cn 说明

本目录是 `docs/architecture`、`docs/archive`、`docs/design`、`docs/migration`、`docs/product`、`docs/ui` 中中文文档的**非权威性人工阅读镜像**，仅用于在 Finder 或编辑器中按中文文件名浏览。

> **权威警告**：AI 代理、实现代理、治理流程和自动化编辑必须使用原始权威文档，不得使用 `docs/zh-cn/**` 作为实施来源。`docs/zh-cn` 继续只是 `NON_AUTHORITATIVE_READING_COPY`。

## 关键规则

- **本目录不是权威来源**。所有产品、设计、架构与实现权威仍保留在原始源路径。
- 镜像中的 Markdown 文件在顶部添加了非权威性提示，并对相对链接进行了重写，以便从镜像位置阅读。
- JSON 镜像文件与原始文件字节一致，未做修改。
- 不要基于本目录进行自动化编辑；修改源文件后需要重新生成镜像。
